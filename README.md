#  Cloud DevOps Project

## 📚 Overview

This project represents the culmination of the **DevOps training at the National Telecommunication Institute (NTI)**, in partnership with **iVolve Technologies**. 

It demonstrates an end-to-end DevOps workflow deployed on **AWS ☁️**, covering the full lifecycle of modern application delivery. The project includes:

- 🛠️ Infrastructure provisioning using **Terraform**
- 🐳 Containerization with **Docker**
- ☸️ Orchestration using **Kubernetes**
- ⚙️ Configuration management with **Ansible**
- 🔧 Continuous Integration with **Jenkins**
- 🚀 Continuous Deployment using **ArgoCD** (GitOps approach)
